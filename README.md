ng-treeview
===========

Angular Treeview UI Widget

```javascript
<div ng-treeview 
    tree-model="treeModel" 
    node-id-key="'id'" 
    node-label-key="'name'" 
    node-children-key="'children'" 
    class="treeView"></div>
```
