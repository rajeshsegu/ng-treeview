ng-treeview
===========

Angular Treeview UI Widget

![ngTreeview](rajeshsegu.github.com/repository/css/ngTreeview.png)


```javascript

<div ng-treeview 
    tree-model="treeModel" 
    node-id-key="'id'" 
    node-label-key="'name'" 
    node-children-key="'children'" 
    class="treeView"
    ></div>

```
